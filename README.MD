# 简介

WeBot 是一款 开源、轻量级的微信自动留言小工具，支持**挂机自动回复**以及**自行拓展插件**的功能，两开花！



# 运行指南



## 刚需：

+ python 3.x.x 的版本
+ itchat, requests 模块



## 软需

+ matplotlib
+ PIL



## 怎么用

1. 运行`chatbot.py`
2. 微信手机扫弹出来的二维码，并授权
3. DONE



## 如果报错

在朋友的电脑上运行并没有报错，如果报错，请先自行百度，实在不行再发issue
